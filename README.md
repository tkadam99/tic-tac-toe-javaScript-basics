
# 🎮 **Tic-Tac-Toe — JavaScript Basics Project**
A simple, clean, and interactive Tic-Tac-Toe game built using HTML, CSS, and vanilla JavaScript. 
This project demonstrates core JavaScript fundamentals including DOM manipulation, event handling, game logic, 
and UI state management.

### **✨ Features**
* Two-player Tic-Tac-Toe gameplay

* Turn-based logic (X and O alternate automatically)

* Win detection for all 8 winning patterns

* Draw detection when all boxes are filled

* Disabled boxes after selection

* "Play Again" button after game ends

* "Reset Game" button during gameplay

* Clean UI with responsive layout

* Fully commented JavaScript code for clarity


### 🧠 **What You Learn From This Project**

* Selecting and manipulating DOM elements

* Handling click events

* Managing game state with variables

* Implementing win/draw logic

* Showing/hiding UI components dynamically

* Resetting UI and game state

* Writing clean, readable, commented code

### 🕹️**How to Play**

* Player X starts the game.

* Click any empty box to place your symbol.

* Players alternate turns automatically.

* The game ends when:

A player wins

All boxes are filled (draw)

* After the game:

"Play Again" appears

"Reset Game" is hidden

During gameplay:

"Reset Game" is available
